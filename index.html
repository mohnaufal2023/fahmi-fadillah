About Me

Hello, saya adalah seorang junior engineer yang berfokus pada pengembangan dalam ranah robotika. Keahlian saya terletak pada merancang dan mengimplementasikan solusi teknis untuk sistem robotika yang kompleks. Saya memiliki pengalaman dalam menggunakan bahasa pemrograman khusus untuk robotika, seperti Python atau C++, serta memahami secara mendalam kerangka kerja robotika seperti ROS (Robot Operating System). Pengalaman saya melibatkan desain dan pengembangan perangkat keras dan perangkat lunak untuk robot, mencakup integrasi sensor-sensor dan pemrograman kontrol motor. Selain itu, saya memiliki keterampilan dalam mendesain algoritma navigasi dan pemrosesan data sensor untuk meningkatkan kinerja robot.
